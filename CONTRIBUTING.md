# Contributing to Honeycomb Cell System

Thank you for your interest in HCS! As an architectural framework, we welcome contributions in the following areas:

1. **Research & Theory:** Formalizing computational complexity and stability analysis.
2. **Runtime Implementations:** Developing engines for HCS execution in Browser, Node.js, or Native environments.
3. **Tooling:** Creating HDA encoders, visualizers, and debuggers.

## Development Process

- Treat "Docs as Code".
- Ensure all technical changes align with the foundational principles in [ARCHITECTURE.md](./ARCHITECTURE.md).
- Use Pull Requests for all changes.

## Code of Conduct

Please be respectful and professional in all interactions.
